#Emulator
A 16 bit emulator with 65k of RAM

##Registers
###General Purpose
- ac
- bc
- dc
- cc
- ec
- fc

### Reserved/Special Purpose
- sp  (stack pointer)``
- pc (Program Counter)
- zo (zero)

##Instruction Set
- load
- store
- skip
- jmp
- halt
- add
- push
- pop
- push
- mul
- li
- sub
- negate
